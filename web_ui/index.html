📖 使用流程

  1. 粘贴机场订阅链接或 sublinkPro 生成的订阅链接到上方输入框
  2. 点击 "刷新订阅"，解析 vmess / vless / ss / trojan / hysteria2 等节点并生成 sing-box 配置
  3. 确认上方“运行诊断”中的配置文件状态、最近状态与日志尾部无异常
  4. 打开 "已启用" 开关并保存配置
  5. 点击 "启动 sing-box"，注册机将自动通过 sing-box 本地混合代理入口出站
